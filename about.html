About & Future Scope

This project showcases the baseline features of a student portal using only HTML, CSS, and JavaScript.

Tech stack

  • HTML for structure
  • CSS for styling
  • JavaScript for data rendering, filtering, and search

Future scope

  • Authentication (student login with email/college ID)
  • Real-time chat or messaging for Connect
  • Sell/Request forms with backend storage
  • Admin dashboards and moderation tools
  • Responsive improvements and accessibility features
Built for demo and extension in future sprints
